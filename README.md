# Credit Card validator using Luhn Checksum Algorithm

Do the following commands
- npm install
- npm run dev (to start)
- npm run test (to run test scripts)

Tech Stack
1. React
2. Typescript
3. Remix (used for ease of setting up node server)
4. Tailwind
